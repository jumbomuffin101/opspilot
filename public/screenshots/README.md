# OpsPilot Screenshot Assets

Place portfolio screenshots here after exporting optimized PNG or WebP files.

Suggested filenames:

- `homepage.png` - landing page hero and product positioning.
- `slack-agent.png` - Slack Agent Experience with suggested prompts.
- `incident-investigation.png` - OpsPilot incident brief in Slack.
- `repository-audit.png` - repository audit result.
- `incident-room.png` - generated incident channel and checklist.
- `postmortem.png` - generated postmortem draft.
- `setup-github.png` - GitHub repository picker or setup flow.

Keep screenshots compressed and avoid exposing real tokens, private repository names, customer data, or private Slack messages.
