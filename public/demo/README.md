# OpsPilot Demo Media

Place lightweight public demo assets here.

Recommended files:

- `opspilot-demo-thumbnail.png` - compressed thumbnail or poster frame for the README.
- `opspilot-demo-preview.mp4` - optional short compressed preview clip only if it is small enough for the repository.

Do not commit the full final demo video if it is large. The current local demo video is approximately 265 MB, so the README should link to a hosted YouTube, Loom, Devpost, or release asset instead.
